Conversor de Favicon para FeedbackAtivo

Instruções:

  1. Clique nos botões abaixo para gerar cada versão do favicon
  2. Clique com o botão direito nas imagens geradas e selecione "Salvar imagem como..."
  3. Salve cada uma com o nome apropriado na pasta public:
    • favicon.png
    • apple-touch-icon.png
  4. Para o favicon.ico, use um conversor online como favicon.io para converter o PNG para ICO
